HTML은 웹 페이지의 뼈대

HTML이란 무엇일까요?

HTML(HyperText Markup Language)은 웹 페이지의 콘텐츠와 구조를 정의하는 표준 마크업 언어입니다. 텍스트, 이미지, 링크 등의 요소를 사용하여 웹 페이지의 뼈대를 만듭니다. '하이퍼텍스트'는 다른 페이지로 이동할 수 있는 링크를 의미하며, '마크업'은 태그(< >)를 사용하여 콘텐츠의 역할이나 구조를 표시하는 것을 의미합니다.

HTML5의 주요 특징

HTML5는 최신 HTML 표준으로, 웹 개발에 여러 가지 강력한 기능을 추가한 버전입니다. 비디오 및 오디오를 위한 멀티미디어 지원이 내장되었고, 그래픽을 위한 <canvas> 및 <svg> 요소가 도입되었습니다. 또한, <header>, <footer>, <section>과 같은 시맨틱 태그를 통해 웹 페이지의 구조를 더 의미론적으로 만들 수 있게 되었습니다.
